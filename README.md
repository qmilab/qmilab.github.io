# QMI Lab — Quantum & Machine Intelligence Laboratory

Website for [QMI Lab](https://qmilab.com), an independent research lab studying intelligence, learning, and representation across classical and quantum computation.

## Research Pillars

- **Foundations of Machine Intelligence** — Representation, cross-lingual transfer, world-model evaluation, and learning dynamics.
- **Quantum Machine Intelligence** — Hybrid quantum-classical architectures, matched-resource benchmarking, and honest tests of quantum advantage.
- **Quantum World Models** — Quantum simulation, world-model structure, and evaluation in physically grounded domains.

## Development

```bash
npm install          # Install dependencies
npm run dev          # Start dev server (localhost:4321)
npm run build        # Build static site to dist/
npm run preview      # Preview built site locally
```

Built with [Astro](https://astro.build).
